Iconos animados

Idle siempre activo · Tocá cada uno para el efecto extra

Cómo usarlos en el proyecto

  1. Copiá icons.css y icons.js a tu proyecto.
  2. Enlazá el CSS y el JS en tu HTML.
  3. Copiá el <button class="app-icon …"> del icono que necesites (incluye el SVG).
  4. Clases: app-icon--fire, app-icon--coin, app-icon--trophy.
