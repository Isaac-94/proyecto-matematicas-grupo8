***- (WebDb/Mock) se define al usar los comandos de inicio: `pnpm dev:all` o `pnpm dev:all-mock` pero el .env requiere distinta configuración para cada caso de uso.***

### - Incluir los `.env` en `.gitignore`

# Usar solo las variables de una "Versión", comentar la otra:

## - Versión `.env` Front-End para "Db" y "SignIn/SignUp" con "Mock" en servidor local:

DATA_SOURCE=MOCK
VITE_API_URL="http://127.0.0.1:3001/api"

## - Versión: `.env` Front-End para conectar con WebDb y Auth (SignIn/SignUp) desde servidor local:

DATA_SOURCE=DB
VITE_API_URL=https://[TU_PROYECTO].vercel.app/api

# Configuración de Supabase (Auth de cliente) desde local o Vercel, para Mock dejar [PLACEHOLDERS]
# Deben coincidir con SUPABASE_URL y SUPABASE_ANON_KEY del Back-End
VITE_SUPABASE_URL="https://[TU_PROYECTO].supabase.co"
VITE_SUPABASE_ANON_KEY="[ANON_KEY]"
